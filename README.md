# yup-cat

```
NAME:
   cat - concatenate files and print on the standard output

USAGE:
   cat [OPTIONS] [FILE...]

      Concatenate FILE(s) to standard output.
      With no FILE, or when FILE is -, read standard input.

COMMANDS:
   help, h  Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --number, -n         number all output lines (default: false)
   --show-ends, -E      display $ at end of each line (default: false)
   --show-tabs, -T      display TAB characters as ^I (default: false)
   --squeeze-blank, -s  suppress repeated empty output lines (default: false)
   --trim-spaces        trim trailing spaces from lines (default: false)
   --help, -h           show help
```
